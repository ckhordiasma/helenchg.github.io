Telescope Module Interface


MEMS designed using LASI7

Project Overview

This senior design project was in conjunction with the Oakley Observatory on campus and its advisor Dr. Richard Ditteon of the Rose-Hulman Physics and Optics Department. Specifically, the project  focused on one of the Richey-Chretien telescopes in the Observatory and its control system, the Telescope Interface Module (T.I.M.). The important fact is that the telescope is unusable without this control box to manage it. The T.I.M. is prone to failure and although there are drop in solutions in the market, they are costly and have features that the clients does not need. The goal of the project was to design a more economical control box that like the original, comes with a controlling software.   The final version of our solution uses new an Arduino Mega as the brain of the replacement control box. It has the ability to turn the cooling fans, sense temperatures of the ambient air, the primary mirror housing and the secondary mirror housing, and control the primary mirror motor. In order to tie it together, the team built a Graphical User Interface (GUI) that allows remote control of the telescope. My main tasks in this project was to work on designing the hardware for the replacement control box, building the GUI, and interconnecting the hardware with the software, while my teammates worked on reverse engineering the original control box.
